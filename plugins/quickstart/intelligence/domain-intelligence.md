# Domain Intelligence - Industry & Technology Expertise

This module provides domain-specific knowledge that helps QuickStart create contextually appropriate extensions for different industries and technology ecosystems.

## Healthcare Domain

**Regulatory Requirements:**
- HIPAA compliance mandatory for patient data
- Audit trails for all data access
- Encryption at rest and in transit
- Access controls and authentication

**Common Patterns:**
- Patient data handling with PHI protection
- Electronic Health Records (EHR) integration
- Clinical workflow automation
- Reporting with de-identification

**Technology Stack:**
- HL7/FHIR standards
- Secure databases (encrypted)
- Compliance-focused logging

## Financial Services Domain

**Regulatory Requirements:**
- SOX compliance for publicly traded companies
- PCI DSS for payment data
- SEC regulations for trading
- AML/KYC requirements

**Common Patterns:**
- Precision calculations (Decimal, not float)
- Audit trails for all transactions
- Real-time risk monitoring
- Regulatory reporting

**Technology Stack:**
- High-precision mathematics libraries
- Secure API integrations
- Transaction databases

## E-Commerce Domain

**Key Requirements:**
- Inventory management
- Order processing workflows
- Customer data handling
- Payment integration

**Common Patterns:**
- Product catalog management
- Cart and checkout flows
- Shipping integrations
- Customer analytics

**Technology Stack:**
- Payment gateways (Stripe, PayPal)
- Shipping APIs
- Database for inventory

## SaaS/Startup Domain

**Key Characteristics:**
- Move fast, iterate quickly
- Limited resources
- Rapid scaling needs
- Flexibility over robustness initially

**Common Patterns:**
- MVP development
- Metrics tracking (MRR, churn, etc.)
- User onboarding automation
- Growth analytics

**Technology Stack:**
- Cloud-native (AWS, GCP, Azure)
- Analytics (Mixpanel, Amplitude)
- Payment (Stripe)

## DevOps Domain

**Key Requirements:**
- CI/CD automation
- Infrastructure as code
- Monitoring and alerting
- Deployment automation

**Common Patterns:**
- Build pipelines
- Automated testing
- Deployment strategies
- Monitoring dashboards

**Technology Stack:**
- Docker, Kubernetes
- GitHub Actions, Jenkins
- Terraform, Ansible
- Prometheus, Grafana

## Data Science Domain

**Key Requirements:**
- Data cleaning and preparation
- Analysis and modeling
- Visualization
- Reproducibility

**Common Patterns:**
- Jupyter notebooks
- Data pipelines
- Model training and evaluation
- Result visualization

**Technology Stack:**
- Python (Pandas, NumPy, scikit-learn)
- Jupyter
- Visualization (Matplotlib, Seaborn)

This intelligence informs context-appropriate generation across all QuickStart modules.
