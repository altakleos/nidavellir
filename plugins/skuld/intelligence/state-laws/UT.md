# Utah (UT) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Common Law
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard or holographic |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Statutory form available |
| Healthcare | N/A | Required | Advance healthcare directive |

## Key Features

### No State Death Taxes
- No estate tax
- No inheritance tax
- Tax-friendly state

### Holographic Wills
- Utah recognizes holographic wills
- Material provisions in testator's handwriting
- Signed by testator

### Utah Uniform Probate Code
- Modern UPC state
- Simplified probate procedures
- Independent administration preferred

### Transfer on Death Deed
- Utah has TOD deed for real estate
- Avoids probate
- Revocable during lifetime

## Elective Share
- Surviving spouse: varies by length of marriage
- Up to 50% for marriages of 15+ years
- UPC-based augmented estate

## Common Considerations
1. UPC state - modern probate code
2. Holographic wills valid
3. TOD deed available
4. Large LDS (Mormon) community - may have specific traditions
5. Consider family size in estate planning (often larger families)
