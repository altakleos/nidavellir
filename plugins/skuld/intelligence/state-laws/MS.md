# Mississippi (MS) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Common Law
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard or holographic |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Durable POA |
| Healthcare | 2 | Not required | Advance healthcare directive |

## Key Features

### No State Death Taxes
- No estate tax
- No inheritance tax
- Tax-friendly state

### Holographic Wills
- Mississippi recognizes holographic wills
- Entirely in testator's handwriting
- Signed by testator
- No witnesses required

### Tenancy by the Entirety
- Available for real property
- Married couples only

## Elective Share
- Surviving spouse entitled to:
- 1/2 of estate if no children
- Child's share if children (at least 1/4)

## Common Considerations
1. No state death taxes
2. Holographic wills valid but formal will recommended
3. Simple probate process
4. Consider trust for larger estates
