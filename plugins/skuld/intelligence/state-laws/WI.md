# Wisconsin (WI) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Marital Property (Community Property variant)
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard execution |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Statutory form available |
| Healthcare | 2 | Not required | Power of attorney for healthcare |

## Key Features

### Marital Property State
- Based on Uniform Marital Property Act (UMPA)
- Substantially similar to community property
- Property acquired during marriage is "marital property"
- Each spouse owns 50%
- Double step-up in basis at death

### No State Death Taxes
- No estate tax
- No inheritance tax
- Tax-friendly state

### Transfer on Death Deed
- Wisconsin has TOD deed for real estate
- Avoids probate
- Revocable during lifetime

### Marital Property Agreement
- Spouses can modify property classification
- Prenuptial and postnuptial agreements

## Elective Share
- Traditional elective share replaced by marital property protections
- Surviving spouse owns 50% of marital property automatically

## Common Considerations
1. Marital property state (like community property)
2. Double step-up in basis available
3. TOD deed available for real estate
4. Unique UMPA terminology
5. Consider IL coordination for Milwaukee area
