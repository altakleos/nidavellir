## Generator Usage

When generating this checklist, the agent should:

1. **Pre-populate** all known information from client profile
2. **List all placeholders** marked [[ ATTORNEY REVIEW ]] from documents
3. **Include state-specific** compliance items
4. **Flag complexity indicators** that were triggered
5. **Summarize key decisions** the client made during discovery

Output format:
```yaml
document_type: attorney-review-checklist
client: "[Name]"
state: "[State]"
document_content: |
  [Populated checklist]

documents_included:
  - revocable-living-trust
  - pour-over-will-client
  - etc.

attorney_review_items:
  - item: "Verify trust name format"
    document: "revocable-living-trust"
    line: 15
  - etc.

complexity_flags:
  - "special_needs_beneficiary"
  - etc.
```
