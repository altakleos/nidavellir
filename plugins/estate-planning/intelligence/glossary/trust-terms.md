# Trust-Related Terms

Load when generating trust documents.

| Term | Definition |
|------|------------|
| **Revocable Trust** | Trust that grantor can modify or revoke during lifetime. Becomes irrevocable at death. Primary tool for probate avoidance. |
| **Irrevocable Trust** | Trust that generally cannot be modified once created. Used for asset protection, tax planning, and special needs planning. |
| **Living Trust** | Trust created during grantor's lifetime (as opposed to testamentary trust created by will). Can be revocable or irrevocable. |
| **Testamentary Trust** | Trust created by will that comes into existence only at death. Must go through probate. |
| **Trust Funding** | Process of transferring assets into trust ownership. An unfunded trust provides no probate avoidance. |
| **Trust Corpus/Principal** | The assets held in the trust. Distinguished from income generated by those assets. |
| **Trust Protector** | Third party with power to modify trust terms, remove trustees, or adapt to legal changes. Provides flexibility. |
| **Incapacity Provisions** | Trust provisions governing what happens if grantor becomes incapacitated. Successor trustee takes over management. |
| **A-B Trust** | Estate tax planning structure dividing trust at first death into "A" (survivor's) and "B" (bypass/credit shelter) trusts. Less common after portability. |
| **Bypass Trust** | Trust designed to use deceased spouse's estate tax exemption. Assets pass to children at survivor's death without being in survivor's estate. |
| **QTIP Trust** | Qualified Terminable Interest Property trust. Provides for surviving spouse while controlling ultimate distribution. Common in second marriages. |
| **Crummey Powers** | Withdrawal rights given to trust beneficiaries to qualify gifts for annual exclusion. Named after court case. |
| **Trust Situs** | State whose laws govern the trust. Can sometimes be changed for tax or asset protection benefits. |
