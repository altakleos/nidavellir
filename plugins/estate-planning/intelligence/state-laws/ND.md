# North Dakota (ND) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Common Law
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard execution |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Statutory form available |
| Healthcare | 2 | Not required | Healthcare directive |

## Key Features

### No State Death Taxes
- No estate tax
- No inheritance tax
- Tax-friendly state

### Transfer on Death Deed
- North Dakota has TOD deed for real estate
- Avoids probate
- Revocable during lifetime

### UPC State
- Adopted Uniform Probate Code
- Modern probate procedures
- Simplified administration

## Elective Share
- Surviving spouse: varies by length of marriage
- Up to 50% for marriages of 15+ years
- UPC-based augmented estate

## Common Considerations
1. UPC state - modern probate code
2. TOD deed available for real estate
3. Large agricultural holdings - special succession planning
4. Oil/mineral rights considerations in some areas
