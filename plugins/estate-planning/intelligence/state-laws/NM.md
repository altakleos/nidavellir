# New Mexico (NM) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Community Property
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard execution |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Statutory form available |
| Healthcare | 1 | Not required | Healthcare decisions |

## Key Features

### Community Property State
- Property acquired during marriage is community
- Each spouse owns 50%
- Double step-up in basis at death
- Quasi-community property rules for property acquired elsewhere

### No State Death Taxes
- No estate tax
- No inheritance tax
- Tax-friendly state

### Transfer on Death Deed
- New Mexico has TOD deed for real estate
- Avoids probate
- Revocable during lifetime

## Elective Share
- No traditional elective share (community property state)
- Community property protections instead

## Common Considerations
1. Community property state - double step-up available
2. TOD deed available for real estate
3. Only 1 witness required for healthcare directive
4. Coordinate with TX if border area
