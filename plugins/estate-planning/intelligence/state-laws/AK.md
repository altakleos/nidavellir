# Alaska (AK) - Estate Planning Requirements

**Tier:** 2 (Standard Documentation)
**Property Type:** Common Law (with Community Property Trust option)
**State Tax:** None

## Execution Requirements

| Document | Witnesses | Notarization | Special Requirements |
|----------|-----------|--------------|---------------------|
| Will | 2 | Self-proving available | Standard execution |
| Trust | N/A | Required | Notary acknowledgment |
| POA | N/A | Required | Statutory form available |
| Healthcare | N/A | Required | Advance healthcare directive |

## Key Features

### Community Property Trust Option
- Alaska allows opt-in community property treatment
- Married couples can elect CP for trust assets
- Provides double step-up in basis benefit
- Must create Alaska Community Property Trust

### Asset Protection Trust (DAPT)
- Alaska was pioneer in domestic asset protection trusts (1997)
- 10-year statute of limitations for fraudulent transfers
- Self-settled trusts can have spendthrift protection
- Must have Alaska trustee and some Alaska assets

### No State Death Taxes
- No estate tax
- No inheritance tax
- Only federal taxes apply

## Elective Share
- 1/3 of augmented estate
- Includes nonprobate assets

## Common Considerations
1. Community property trust for step-up benefits
2. Asset protection trust opportunities
3. Remote state - consider accessibility of trustee
4. Unique native corporation land issues
